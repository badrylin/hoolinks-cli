# Hoolinks-cli

昊链专用脚手架工具

## 功能点

- [x] 支持多模块，多入口，
- [x] 支持参数化构建
- [x] 支持自定义多环境打包
- [x] 打包模块体积分析
- [x] 打包速度分析
- [x] 支持css modules, less
- [x] 支持CDN
- [x] 支持typescript
- [ ] 支持tslint
- [x] DLL
- [x] 多线程
